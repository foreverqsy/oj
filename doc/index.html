• 1如何报名？报名时有什么需要注意的？

  •       进入acm.sut.edu.cn单击【报名参赛】后，在转到网页的右上角点击【登录|注册】，滑出菜单的左栏为注册页面。

          增加队员请按下输入框左侧的【加号】，在填写完队员信息后请选择您要参加的比赛类型。“新生赛”仅限13级同学报名，而“校赛”则全校同学均可报名但不可兼报。之后请输入您的登陆密码，在比赛前，您的新密码将会以密封的形式送到您手上。

          报名时如遇到任何问题，欢迎向平台管理员进行咨询。

  • 2我在注册的时候不小心把信息输入错误了，请问可以更改吗？

  •       可以，请在登陆后点击页面右上角的“修改信息”来更改您和队友的信息。

          *新生赛和校赛比赛类别在报名后不可更改，如需更改请在报名时间段内联系平台管理员并提交队长和队员信息材料进行人工更改。

  • 3忘记报名号、忘记密码怎么办？

  •       报名后请牢记您的报名号，即作为比赛时的登录账号，如忘记请点击SUTOJ首页的【参赛队】，根据您队伍名找回报名号

          密码仅供举办方进行录入资料所用，如忘记也无需找回，在比赛前，您的新密码将会以密封的形式送到您手上。

  • 4能否重新注册队伍？如何退队、更换队伍？

  •       不可以，所有参赛者的学号只能存在于一个队伍中，如要把队友请出队伍、自愿退出队伍或是更换队伍，请登录后点击右上角的【修改信息】，将队友或您的信息删除即可，然后加入其它队伍即可。

  • 7新生赛样题

  • A:数列

          Description

         这里有一个等差数列a = {1,2,3,4,5，……，100},求出数列的总和。

     

          Input 

    提交答案的方法如下：

    首先在题目下找到“submit”按钮，点击进入后将以下代码复制到页面中间的输入框中。

    #include<stdio.h>

    int main()

    {

    printf("答案\n");

    return 0;

    }

    最后将你算出的答案替换代码中的“答案”两字，再次点击下面的“submit”按钮即可完成提交。

          样例答题方法

     

    答案为：5050

    提交的代码是：

    #include<stdio.h>

    int main()

    {

    printf("5050\n");

    return 0;

    }
