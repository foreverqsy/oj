OJ使用及参赛说明

1.OJ开放时间即比赛时间为5个小时，热身赛为2个小时

2.OJ支持C或C++语言

3.通过 acm.sut.edu.cn 访问“SUT Online Judge”,使用报名号和密码登陆。报名号和密码将会严密封装在档案袋中。

4.竞赛开始，可以通过“比赛题目”界面查看题目

5.选择题目，编写程序并通过“Submit”按钮提交代码，注意选择对应的题号和语言（提交C程序选GCC，提交C++程序选G++）

6.通过“Submit”界面提交代码后，自动跳转到“提交情况”界面，即可查看评测结果。

7.在“现场排名”界面可查看即时排名。

8.判决结果由在线评测系统自动反馈给参赛选手。如果正确，就为该选手挂上一个气球，不同颜色的气球代表不同的题目。

9.参赛选手可能收到的反馈信息包括：

    Ø pending: 系统正在做评判的准备工作，可以使用“刷新”来查看。

        Ø Accepted: 你的解答是正确的，恭喜！！

        Ø Presentation Error: 输出时表达错误，如果看到这样的结果，说明你的程序已经基本正确了，只是多打了换行符或多输出了空格，检查一下你的程序和题目要求吧，胜利就在眼前！

        Ø Wrong Answer: 你的程序正常运行并输出结果，不过是答案错误……

        Ø Runtime Error: 你的程序正常编译，不过运行时发生错误，请检查是否数组越界，指针操作违法……

        Ø Time Limit Exceeded: 在题目规定的时间内你的程序没能输出答案；

        Ø Memory Limit Exceeded: 你的程序申请了太多的内存，超出题目限制；

        Ø Compile Error: 编译错误，请查看系统反馈的编译信息。

只有评测结果为“Accepted”的题才算正确，其余结果均为错误。

10.参赛选手首先根据解题数目进行排名。最后的获胜者为正确解答题目最多且总用时最少的队伍。每道试题用时将从竞赛开始到试题解答被判定为正确为止，其间每一次提交运行结果被判错误的话将被加罚20分钟时间，未正确解答的试题不记时。

        在决定胜出选手时，如果多位选手解题数量相同，则根据总用时加上惩罚时间进行排名。总用时和惩罚时间由每道解答正确的试题的用时加上惩罚时间而成。每道试题用时将从竞赛开始到试题解答被判定为正确为止，其间每一次错误的运行将被加罚20分钟时间，未正确解答的试题不计时。

        例如：A、B两队都正确完成两道题目，其中A队提交这两题的时间分别是比赛开始后1:00和2:45，B队为1:20和2:00，但B队有一题提交了2次。这样A队的总用时为1:00+2:45=3:45而B队为1:20+2:00+0:20=3:40，所以B队以总用时少而获胜。

参赛说明

1.参赛选手有权提交解释请求，针对题目描述中的不明确或错误的部分提问。如果评委确认题目中确实存在不明确或错误的部分，将会通告所有参赛选手进行声明或更正。

2.每位选手使用一台计算机，所有选手使用计算机的规格配置相同且设备环境指纹均相同

用户环境

竞赛时，选手所使用的计算机为如下环境：

Windows XP

3.参赛选手可以携带诸如书、手册、程序清单或源码等纸质参考资料

4.参赛选手不能携带任何可用计算机处理的软件或数据（不允许任何私人携带的磁盘和计算器）

5.参赛选手不能携带任何通讯工具，包括无线电接收设备、移动电话等。

6.在竞赛中，参赛选手不得通过网络工具（如E-mail、BBS、QQ以及所有SNS服务等）以及任何其他方式和参赛选手以及竞赛组委会指定工作人员以外的人交谈。

7.竞赛的预定时间为5个小时热身赛为2个小时，但当竞赛进行到一定时间后，竞赛组委会可以因为出现不可预见的时间而调整比赛时间长度，一旦比赛时间长度发生改变，将会以及时并且统一的方式通告所有参赛选手。

8.但竞赛选手出现妨碍比赛正常进行的行为时，诸如擅自移动、更改赛场中的设备或电力设施，未经授权修改比赛软硬件，干扰他人比赛等，都会被竞赛组委会剥夺参赛资格。

9.本次比赛在SAC(SUT Anti-cheating)反作弊系统的监视下进行，将会在您登陆时收集您的设备环境指纹信息，请勿在组委会指定的设备外进行登录、答题操作。
